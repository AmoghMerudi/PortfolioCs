Hardware Project

In the hardware project, I was provided with various tasks which can be performed by a device and had to explain one of the processes in detail. I chose to work on the topic "The user has a basic painting application open and taps on the screen to draw a point". Here is a link which will take you to my Hardware Project.

Reflection

This project has significantly helped me in understanding the function and synchronization of the computer systems and components to fire up multiple pixels to display an output on the screen of a phone. I also learnt about different parts of the GPU which I did not know existed prior to this assignment.

On this topic, I still must learn more about the various components of a electronic device. Since my research was mainly focused on phone components, my understandings on the same components in a laptop or a personal computer are not quite clear.

How might this hardware project have imporved your understanding of computer components? Explain using an example of hardware components.
Before I did this project, I thought that the GPU was one singular unit which renders the output on the screen. During my research, i found out about various parts of the GPU which include:

  1. VRAM, which stands for Video Random Access Memory, allows the GPU to transmit data to different internal components of the phone.
  2. VRM, which stands for Voltage Regulator Module, regulates the voltage current recieved by the GPU and other parts of the graphics card by the battery. This charge is regulated to avoid any surge taking place in the graphics card.
